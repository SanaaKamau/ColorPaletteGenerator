# Color Palette Generator

A simple website that allows users to generate random color palettes, add or remove colors, and regenerate palettes.

## Features

- Add new colors with the `+` button
- Remove colors with the `-` button
- Regenerate all colors
- Display colors as hex codes
- Horizontal layout of color cards
